## Why StarMap AI
- ‚ö° Faster: ask once, multiple models respond in parallel
- üß≠ Safer: compare consensus vs disagreements to reduce single-model risk
- üß© Cleaner workflow: one dashboard instead of tab-hopping
- üîí Privacy-first: no data collection, no server uploads, local aggregation only

## Highlights
- Multi-site dashboard for AI models and search engines
- One-click send to selected models
- One-click export: merge model outputs into a Markdown file (search engines excluded)
- Extensible: add/update sites via `sites.js`

## Install (Developer Mode)
1. Open extensions page  
   - Chrome: `chrome://extensions/`  
   - Edge: `edge://extensions/`
2. Enable Developer Mode
3. Click ‚ÄúLoad unpacked‚Äù
4. Select the `starmap-ai-extension` folder

## Usage
1. Click the extension icon to open the dashboard
2. Select models in Settings
3. Type your prompt and click Send
4. (Optional) Click ‚ÄúSave outputs‚Äù to export Markdown
